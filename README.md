# PRISM

> [cite_start]Provenance-backed Reproducible Intervention-response Scoring of Mechanisms [cite: 2]

## 概要
[cite_start]静的に区別不能な機構を、介入というプリズムでスペクトルに分離する評価装置です。 [cite: 3]
[cite_start]任意の金融 ABM を、既知の構造変更を伴う実市場の自然実験コーパス（Tick Size Pilot等）に通し、「モデルが予測する stylized facts の変化量△F」を「実データで観測された変化量 △F」と照合して、機構の正しさを採点します。 [cite: 5, 10]

[cite_start]本プロジェクトは、出力として順位表ではなく、完全再現可能・監査可能な「位相図テンソル (機構族 × 介入クラス × fact → 一致度)」を生成します。 [cite: 5]
