金融マーケット ABM — 実験 playground

YH001〜YH006 をブラウザで触る。Cont-Bouchaud から Speculation Game まで。

各ページはリポジトリの 本物の model.py / simulate.py を Pyodide(ブラウザ内 Python)で実行し、パラメータをスライダで変えて結果プロットを即更新する。コードは複製せず、実体ファイルを fetch して走らせている(single source of truth)。

起動方法: リポジトリ直下で
python -m http.server 8000
を実行し、http://localhost:8000/playground/index.html を開く。
(ファイルを直接ダブルクリックすると CORS で model.py を fetch できない。Pyodide 自体は CDN から取得するのでネット接続が要る。)

設計メモ / 注意
  • 忠実度: JS 再実装ではなく実 Python を走らせるので Python 版と挙動一致。ただしブラウザ負荷を抑えるため N/T は論文より小さいデフォルト。点推定(Hill α 等)は単一 trial で揺らぐ。
  • 重い処理: YH001 は scipy ロードで初回が重い。YH005/005_1 は per-close ループがあり T を上げると数十秒。論文の重い sweep(powerlaw MLE / 多 trial)はブラウザ版では省略 or 縮小し、numpy で代替できる軽量版に置換。
  • YH006 系: PAMS ライブラリ + parquet データ依存でブラウザ実行不可。生成済み図の結果ビューア。aggregate 側の機構は YH005_1 で interactive に再現できる。
