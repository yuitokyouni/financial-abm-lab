YH002 · Lux & Marchesi (2000) Volatility Clustering

Volatility clustering in financial markets: a microsimulation of interacting agents — IJTAF 3(4) · 実体: experiments/YH002/model.py (Parameter Set I)

楽観/悲観 chartist と fundamentalist の間を、価格トレンド・乖離に応じて確率的に遷移する。chartist 比率 z(t) が臨界値 z̄ を跨ぐ on-off intermittency が volatility clustering と heavy tail を生む。β(価格反応)と σ_μ(ファンダ雑音)を変えて挙動を見る。

エンジン: 本物の experiments/YH002/model.py を numpy で実行。1 integer step = 100 内部ステップなので、step 数を上げると重い。ACF は statsmodels の代わりに同等の numpy 実装で計算。

このモデルについて / 既知の数値
  • z̄ ≈ 0.660 (Appendix A の分岐条件を解いた値、論文本文 0.65 と一致)。z(t) が z̄ 付近で intermittent に振動するとボラが clustering する。
  • Hill tail index: 10% tail で α̂≈1.92 (論文中央値 1.93)。raw return の ACF は lag 1-2 に小さな負スパイク、squared/|r| は緩やか減衰(findings 記載済)。
  • excess kurtosis は単一 seed の最大 burst 頻度に強依存。論文 Table 2 の 135.73 は中央値ベースで、短い run では届かない。
  • ADF はここでは省略(T が長いと検出力が高く unit root を棄却、論文の non-rejection は 500-obs subsample 由来)。
